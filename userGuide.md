# Guia de Uso - LPMS\n\n## Bem-vindo ao Sistema de Gestão de Escritório de Advocacia\n\n**URL:** [Seu link será fornecido após hospedagem]\n\n**Propósito:** Gerenciar processos jurídicos de servidores públicos com automação inteligente, controle de prazos e descoberta de oportunidades.\n\n**Acesso:** Login obrigatório para equipe do escritório | Página pública para clientes\n\n---\n\n## Powered by Manus\n\nEste sistema foi desenvolvido com tecnologias de ponta:\n\n**Frontend:** React 19 + TypeScript + CSS3 Responsivo\n**Backend:** Express.js + Prisma ORM + PostgreSQL\n**IA:** Integração com OpenAI para análise documental e geração de petições\n**Hospedagem:** Auto-scaling em infraestrutura global com CDN\n**Segurança:** Autenticação JWT, criptografia de senhas, HTTPS\n\n---\n\n## Usando Seu Website\n\n### 1. Dashboard - Visão Geral Completa\n\nAo fazer login, você vê um painel com:\n\n- **Total de Clientes:** Número de clientes cadastrados\n- **Total de Processos:** Quantidade de ações em andamento\n- **Prazos Vencidos:** Alertas em vermelho para prazos perdidos\n- **Oportunidades:** Novas teses identificadas para clientes\n- **Receita Prevista:** Estimativa de RPVs a receber\n\n**Ação:** Clique em qualquer card para acessar detalhes ou criar novo registro.\n\n### 2. Onboarding de Clientes - Captura Automática\n\nCompartilhe o link público de onboarding com novos clientes:\n\n1. Cliente acessa o link\n2. \"Preenche dados pessoais\" (nome, CPF, endereço)\n3. \"Clica em Próximo\" e preenche dados profissionais\n4. \"Baixa os 3 PDFs\" (Procuração, Contrato, Declaração de Hipossuficiência)\n5. \"Assina digitalmente\" usando certificado digital\n6. \"Faz upload dos documentos assinados\"\n7. Sistema registra automaticamente no banco de dados\n\n**Resultado:** Novo cliente pronto para ajuizamento de ações.\n\n### 3. Automação de Petições - Geração em Minutos\n\nAcesse \"Automação de Petições\" para gerar peças processuais:\n\n1. \"Selecione o tipo\" (Réplica, Recurso, Contra-razão, Embargo)\n2. \"Escolha a tese jurídica\" (Licença-Prêmio, Abono Permanência, etc)\n3. \"Preencha cliente e número do processo\"\n4. \"Clique em Gerar Petição\"\n5. IA analisa e gera petição em segundos\n6. \"Revise o conteúdo\" e personalize se necessário\n7. \"Baixe o PDF\" pronto para protocolo\n\n**Economia:** De 2-3 horas por petição para 5-10 minutos.\n\n### 4. Descoberta de Oportunidades - Prospecting Automático\n\nIdentifique novas teses para clientes existentes:\n\n1. \"Acesse Descoberta de Oportunidades\"\n2. \"Selecione um cliente\" ou \"Clique em Analisar Todos\"\n3. Sistema analisa dados do cliente (tempo de serviço, cargo, etc)\n4. \"Veja as oportunidades identificadas\" com nível de confiança\n5. \"Filtre por confiança\" (95%, 88%, 72%, etc)\n6. \"Clique em Ajuizar Ação\"\" para processos aplicáveis\n7. Sistema cria automaticamente novo processo\n\n**Benefício:** Aumente receita identificando ações que clientes não sabiam que tinham direito.\n\n### 5. Consulta de Andamento - Página Pública para Clientes\n\nClientes podem consultar andamento sem entrar em contato:\n\n1. \"Compartilhe o link público\" com cliente\n2. Cliente \"entra com CPF\" ou \"número do processo\"\n3. Sistema busca automaticamente no TJDFT\n4. \"IA resume o andamento\" em linguagem clara\n5. Cliente vê \"status atual\" e \"próximos prazos\"\n\n**Vantagem:** Reduz 80% das perguntas de clientes sobre andamento.\n\n### 6. Resposta Sistematizada - Comunicação Padronizada\n\nResponda clientes com templates profissionais:\n\n1. \"Clique em Resposta Sistematizada\"\n2. \"Preencha o cliente\" e \"a pergunta dele\"\n3. \"Selecione o tipo\" (Andamento, Documentos, Prazos, Honorários)\n4. \"Escolha o canal\" (WhatsApp, E-mail, SMS)\n5. \"Clique em Gerar Resposta\"\n6. Sistema cria resposta profissional\n7. \"Personalize se necessário\"\n8. \"Clique em Enviar\"\n\n**Resultado:** Resposta profissional enviada em 1 minuto.\n\n---\n\n## Gerenciando Seu Website\n\n### Settings (Configurações)\n\nNo painel de Management UI, acesse **Settings** para:\n\n- **General:** Alterar nome do escritório, logo, visibilidade\n- **Domains:** Personalizar domínio (de xxx.manus.space para seu domínio próprio)\n- **Secrets:** Adicionar/atualizar chaves de API (OpenAI, etc)\n- **Notifications:** Configurar alertas de prazos\n\n### Dashboard (Análise)\n\nAcesse **Dashboard** para:\n\n- Visualizar estatísticas de uso\n- Ver gráficos de processos por tese\n- Acompanhar taxa de sucesso\n- Exportar relatórios\n\n### Database (Dados)\n\nAcesse **Database** para:\n\n- Visualizar e editar clientes\n- Gerenciar processos\n- Consultar histórico de petições\n- Fazer backup de dados\n\n---\n\n## Next Steps\n\n**Comece agora:**\n\n1.
